2. 定位玩法
选择第1位到第5位中的任意位置。
从1到6中任选一个号码。
如果所选号码与选定位置的中奖号码匹配，则视为中奖。
示例：中奖号码：1,* ,* ,* ,* , * ；投注第1位选择号码1，即视为中奖。 english below Dragon/Tiger:

2. Fixed Place
Select places from 1st to 5th.
Select one number from any 1 to 6.
If the selected number and the winning number in the chosen place match, it is considered a win.
